A guide to WP Full Site

WP Full site is a theme built for small businesses that that is tailored towards the basics of gaining customers.

Most WordPress themes are made for blogging. While WP Full Site allows you to blog, the emphasis is placed on getting your visitors to contact you.

How to install WP Full Site

  1. Install the Hybrid theme first. This is required to run the WP Full Site child theme.
  2. Unzip the wpfullsite.zip file (you've probably already done this).
  3. Upload the wpfullsite folder to your /wp-content/themes directory.
  4. Go to Appearance > Themes in your WordPress admin.
  5. Activate the WP Full Site theme.

Using the theme

The theme is a child theme for the Hybrid theme framework, which means you must have Hybrid installed to use WP Full Site. Don't worry. Just because it uses a framework doesn't mean it's complicated. It just means that your site will be a lot more powerful and all you have to do is have Hybrid installed. Everything should work from the moment you have everything installed.

Theme Settings

WP Full Site allows you to enter information into the WordPress backend and then displays it in the theme

Go to Appearance > Themes > Hybrid Settings and scroll to the bottom to see the WP Full Site settings.

You can enter the following information:

  • Phone Number (displays in the header)
  • Email Address (displays in the header)
  • Call to Action
    • Entering any text here will cause the theme to add a large graphic below the navigation as well as the text you've entered.

Theme Support & Community

The most authoritative sources for finding out how to use the theme or do some nifty stuff are the WP Full Site and Hybrid theme tutorials.

Hyrbid also has a vibrant community at the Theme Hybrid Support Forums. If you ever lose the link, you can find it right from your Hybrid Settings page in your WordPress dashboard.
